# 📌 Overview

The **IMG TO TEXT Converter** is a Python-based OCR (Optical Character Recognition) project that extracts text from images using **OpenCV** and **Tesseract OCR**.

This project allows users to:
- 📷 Upload images
- 🔍 Detect text from images
- 📝 Convert image content into editable text
- ⚡ Perform fast and accurate OCR processing

The system demonstrates the implementation of:
- 👁️ Computer Vision
- 🧠 OCR Technology
- ⚡ Real-Time Image Processing

# 🔍 Working Principle

The project uses **OCR (Optical Character Recognition)** technology to detect and extract text from images.

### Process:
- 📷 Capture or upload image
- 👁️ Preprocess image using OpenCV
- ⚡ Apply OCR using Tesseract
- 📝 Extract text from image
- 💻 Display editable text output

# 📥 Installation

## 🔹 Clone Repository

```bash
git clone https://github.com/Monishpv/IMG_TO_TEXT.git
cd IMG_TO_TEXT
```

## 🔹 Install Dependencies

```bash
pip install opencv-python pytesseract numpy
```

## 🔹 Run the Project

```bash
python main.py
```
